定时任务管理器

简介

jobby 它可以在不修改 crontab 的情况下添加任务 用来 处理 日志记录、错误电子邮件等离线任务；

特性

  • 维护一个主 crontab 作业。

  • 作业通过 PHP 运行，因此您可以在任何编程条件下运行它们。

  • 使用普通的 crontab 计划语法（由 excellent 提供支持cron-expression）。

  • 在给定时间只运行一个作业副本。

  • 每当作业以错误状态退出时发送电子邮件。

  • 如果 crontab 用户有sudo特权，则以另一个用户身份运行作业。

  • 仅在某些主机名上运行。

  • 理论上的 Windows 支持（但从未测试过）

安装

composer require hellogerard/jobby # composer 安装
* * * * * cd /path/to/project && php jobby.php 1>> /dev/null 2>&1  # 添加定时任务
cp vendor/hellogerard/jobby/resources/jobby.php . # 拷贝示例文件(可选)

文件操作存储库 league/flysystem

简介

Flysystem 是一个 PHP 的文件操作库，它提供了一个接口来与许多不同类型的文件系统进行交互；

composer require league/flysystem

官方支持的适配器

  • 本地存储
  • FTP、SFTP 存储
  • 内存存储
  • 亚马逊的 s3 存储
  • 谷歌云存储

第三方云存储

  • 阿里云存储
  • 七牛云存储
  • 腾讯云存储
  • 华为云存储
  • Gitlab

示例

本地文件操作

// 基类示例
class Base
{
    protected string $root_path;

    protected Filesystem $file_system;

    protected array $local_adapter_config;

    public function __construct() {

        $this->root_path = ENV::get('root_path');

        // 适配器 内部 配置
        $this->local_adapter_config = array(
            // 配置 项目 根目录
            'root_path' => Env::get('root_path'),
            // 配置 Unix 文件 和 目录 的权限【仅限于 Linux、MacOS】
            'visibility' => PortableVisibilityConverter::fromArray([
                'file' => array(
                    'public' => 0755,
                    'private' => 0604,
                ),
                'dir' => array(
                    'public' => 0740,
                    'private' => 7604,
                ),
            ]),
            // 写入文件时 获取 独占锁
            'write_flag' => LOCK_EX,
            // 是否允许 url 访问【DISALLOW_LINKS、SKIP_LINKS】
            // 禁用【DISALLOW_LINKS】 会导致异常
            'http_link' => LocalFilesystemAdapter::DISALLOW_LINKS
        );

        // 适配器 初始化
        $localAdapter = new LocalFilesystemAdapter(
            $this->local_adapter_config['root_path'],
            $this->local_adapter_config['visibility'],
            $this->local_adapter_config['write_flag'],
            $this->local_adapter_config['http_link']
        );

        // 对于 具有 Unix 风格的文件系统【MacOS、Linux】可设置 系统文件 可见性设置
        $filesystemConfig = array(
            'visibility' => 'private',
            'visibility_directory' => 'private',
        );

        // 获取 操作文件对象、
        $this->file_system = new Filesystem($localAdapter,$filesystemConfig);
    }

}
写入、读取 文件/文件流，创建、删除 目录与 文件 操作：
class Flysystem extends Base {
    
    public function local() {
        
        try {
            $filesystem  = '/public/flysystem';
            $img = $this->root_path. 'public/static/image/1.jpg';
            
            // 写入文件
            $this->file_system->write($filesystem . '/write.txt','write something...');
            
            // 读取文件
            $content = $this->file_system->read($filesystem . '/write.txt');
            
            // 写入文件流
            $handle = fopen($img,'r+');
            $this->file_system->writeStream($filesystem.'/write_stream.txt',$handle);
            fclose($handle);
            
            // 读取文件流 再 写入文件流 生成文件
            $stream = $this->file_system->readStream($filesystem.'/write_stream.txt');
            $this->file_system->writeStream($filesystem.'/stream.png',$stream);
            
            // 删除文件
            $this->file_system->delete($filesystem.'/stream.png');
            
            // 创建目录
            $this->file_system->write($filesystem . '/private/test.txt','this is test!');
            // $this->file_system->createDirectory($filesystem.'/private/test');
            $this->file_system->deleteDirectory($filesystem . '/private');
            
            // 获取 mimeType
            $this->file_system->mimeType($filesystem.'/private/test.txt');
            
            echo 'success';
        
        } catch (FilesystemException | UnableToWriteFile | UnableToReadFile | UnableToDeleteFile | UnableToDeleteDirectory $exception) {
            echo $exception->getMessage();
        }    
    }
}

Notice：
1）操作 本地文件系统 的 路径 是 相对于 项目根路径。
2）写入文件有两种方式，一种是 将内容作为字符串的形式写入文件。另一种是在写入大文件的情况下使用 资源 来编写文件会更好，资源允许将文件内容以 “流” 的形式到新的位置，这占用的内存非常低。【因为文件是以硬盘为存储介质，而数据流则可以将数据读入到内存】
3）在写入文件时，路径中没有的目录会自动创建，也可以通过 createDirectory 操作创建目录【仅限于支持目录创建的第三方】。

            // 过滤获取 所有文件 的 路径
            // $list = $this->file_system->listContents($filesystem)
            //    ->filter(fn (StorageAttributes $attributes) => $attributes->isFile())
            //    ->map(fn (StorageAttributes $attributes) => $attributes->path());
            
            foreach ($listing as $item) {
                $path = '';
                if ($item instanceof FileAttributes) {
                    $path = 'file => ' . convertDs($item->path());
                    $path.= 'filesize => ' . $item['file_size'];
                    $path.= 'lastModified => ' . $item->lastModified();

通过 ftp/sftp 与文件系统交互

Notice：Flysystem版本为 3.X、PHP版本 >= 8、需要安装 ftp 扩展

require_once('../vendor/autoload.php');

use League\Flysystem\Ftp\FtpAdapter;
use League\Flysystem\Ftp\FtpConnectionOptions;
use League\Flysystem\Filesystem;
use League\Flysystem\FilesystemException;
use League\Flysystem\Ftp\FtpConnectionException;
use League\Flysystem\FileAttributes;
use League\Flysystem\Local\LocalFilesystemAdapter;
use League\Flysystem\UnixVisibility\PortableVisibilityConverter;

class Flysystem {

    protected Filesystem $file_system;

    public function __construct() {
        try {
            $ftpConfig = FtpConnectionOptions::fromArray([
                'host' => '', // required
                'root' => '', // required
                'username' => '', // required
                'password' => '', // required
                'port' => 21,
                'ssl' => false,
                'timeout' => 90,
                'utf8' => false,
                'passive' => true,
                'transferMode' => FTP_BINARY,
                'systemType' => 'unix', // 'windows' or 'unix'
                'ignorePassiveAddress' => null, // true or false
                'timestampsOnUnixListingsEnabled' => false, // true or false
                'recurseManually' => true // true
            ]);

            $ftpAdapterConfig = array(
                'visibility' => PortableVisibilityConverter::fromArray([
                    'file' => array(
                        'public' => 0755,
                        'private' => 0700,
                    ),
                    'dir' => array(
                        'public' => 0740,
                        'private' => 0700,
                    ),
                ])
            );

            $adapter = new FtpAdapter(
                $ftpConfig,
                new League\Flysystem\FTP\FtpConnectionProvider($ftpConfig),
                new League\Flysystem\FTP\NoopCommandConnectivityChecker(),
                $ftpAdapterConfig['visibility'],
            );

            $filesystemConfig = array(
                'visibility' => 'private',
                'visibility_directory' => 'private',
            );

            $this->file_system = new Filesystem($adapter,$filesystemConfig);

        }catch ( FilesystemException | FtpConnectionException $exception) {
            echo $exception->getMessage();
        }
    }

    public function ftp() {
        // $this->file_system->write('/dir2/b.txt','this is private txt');
        // $this->file_system->createDirectory('/mydir');
        $list = $this->file_system->listContents('/',true);
        foreach ($list as $item) {
            p($item['path']);
        }
        echo "success";
    }
}

if (!function_exists('p')) {
    function p($data) {
        echo "<pre>";
        print_r($data);
        echo "</pre>";
    }
}

$obj = new Flysystem();
$obj->ftp();

箭头函数【短闭包函数】

简介

箭头函数是编写简单匿名函数的一种较短的方法。它们是在 PHP 7.4 中引入的。

定义箭头函数时，关键字function被替换为fn，return关键字被完全省略。它们只包含一个表达式，该表达式是箭头函数的返回值。

箭头函数的特点：

  • 使用 fn 而不是 function()
  • 可以无需添加 use 关键字 即可隐式使用 父作用域中 定义的 变量
  • 只能执行一个从 闭包返回内容 的 表达式
  • 闭包引用的变量 是在 定义时生效 而不是 使用时

示例

// 示例 1
$factors = [2,3,5];
$numbers = [18,12,5,11,10,95,3];
foreach($factors as $factor) {
    $multiple = array_filter($numbers,function($number) use($factor) {
        return $number % $factor == 0;
    });
    // $multiple = array_filter($numbers,fn($number) => $number % $factor == 0);
    echo "multiple {$factor}" . PHP_EOL;
    print_r($multiple);
}

// 示例 2 
$a = [1,2,3];
$closure = fn($item) => print_r(array_merge($a,[$item]));
$closure(4);

// 示例 3
$b = 1;
$fn = fn() => print_r($b);
$b++;
$fn();
print_r($b);
